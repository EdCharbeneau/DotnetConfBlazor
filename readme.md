## Dotnet Conf Focus Blazor Editon
### Authoring Component Libraries

In this repo you will find the example used for my session on Dotnet Conf for creating component libraries.

Included:

- Branches for each step taken on the show. You may start with 00-start and follow along or skip to any section.
- All presentation materials are included even the "script" which I used to walk through each step. Please feel free to use this for your own presentation at your company or meetup group.
- Packages are available on NuGet 
  - [BlazorComponentUtilities](https://www.nuget.org/packages/BlazorComponentUtilities/)
  - [BlazorPro.Spinkit](https://www.nuget.org/packages/BlazorPro.Spinkit/)
  - [Blazor Pro Snippets Extension](https://marketplace.visualstudio.com/items?itemName=EdCharbeneau.blazorpro-snippets)
